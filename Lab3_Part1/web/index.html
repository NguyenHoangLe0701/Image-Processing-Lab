03

Cách hoạt động

1

Tiền xử lý

Chuyển ảnh sang grayscale và resize về 256×256 pixel để chuẩn hóa đầu vào.

2

Biến đổi Wavelet (DWT)

Phân tích ảnh thành 4 sub-band: cA (xấp xỉ), cH (ngang), cV (dọc), cD (chéo) bằng PyWavelets.

3

Tạo mã băm

Lượng tử hóa hệ số cA: giá trị ≥ trung bình → 1, ngược lại → 0. Kết quả là chuỗi 4096 bit.

4

Khoảng cách Hamming

Đếm số bit khác nhau giữa 2 hash. Ít bit khác → ảnh tương tự. Nhiều bit khác → ảnh khác biệt.
